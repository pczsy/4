    工善其事必先利器

    打开setting文件选择Plugins选项
    Ctrl + Alt + S
    File -> Setting
    我的idea是最新版本2019.02 有的和别人界面可能不一样
    主界面(左边那个箭头是外网搜索,右边哪个箭头是已经安装的)

    1.Grep console
    自定义日志颜色，idea控制台可以彩色显示各种级别的log，安装完成后，在console中右键就能打开。

    2.Free Mybatis plugin
    之前用了个收费比较好用,刚熟悉就到期了 然后就没然后了,这个时候Free 来了 别提多nice了
    使用mybatis有个痛点就是当你写完一个(自定义)接口的时候就会发现,费劲吧啦的去找我的xml呢哪个是
    眼巴巴的费死劲结果发现找错了 ps:Free Mybatis Plugin 来了解决了所有困扰
    这是一个Dao接口 点击左下角的方块就可进入到xml(这样看代码是不是更省事一些呢!)

    当你新建一个自定义方法的时候 别急 好的工具已经为我们想到了
    新写一个方法 发现报红了 证明这什么

    使用快捷键Alt+Enter 选择第一个就会自己生成一个xml

    点击上左边箭头快速回到Dao

    3.Lombok
    (@Setter,@Getter,@ToString,@Data)之后，就不需要编写或生成get/set等方法，很大程度上减少了代码量，而且减少了代码维护的负担。
    这个工具还有很多方法别如说 @SLF4J 等等 使用的同时记得字母大小写取值问题 我在翻过船
