    • Redis的安装和配置
    原文出处:本文由博客园博主TOUCH_EKKO提供。
    原文连接:https://www.cnblogs.com/fanqinglu/p/11762165.html

    1.下载Redis

    cd /opt/resources/
wget http://download.redis.io/releases/redis-3.2.11.tar.gz
tar -xf redis-3.2.11.tar.gz -C /opt/software/

    2.安装Redis

    cd /opt/software/redis-3.2.11
make && make install

    3.启动单机版Redis

    #启动   默认端口6379
redis-server

#登陆
redis-cli

#关闭服务
shutdown

#退出客户端
quit

    4.分布式配置Redis

    配置文件redis.conf    主要做如下配置，其它默认不变

    第61行      关闭只本地访问
#bind 127.0.0.1

第80行      关闭保护模式
protected-mode no

第84行      端口号
port 6379

第128行    后台守护进程
daemonize yes

第150行    纪录pid值
pidfile /var/run/redis_6379.pid

第163行    日志文件
logfile "6379redis.log"

第202行    持久化保存策略
save 900 1
save 300 10
save 60 10000

第237行    持久化文件rdb
dbfilename dump6379.rdb

    第237行    持久化文件aof
    appendonly yes
