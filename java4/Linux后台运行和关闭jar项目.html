    • Linux后台运行和关闭jar项目
    原文出处:本文由博客园博主听惊雷提供。
    原文连接:https://www.cnblogs.com/share-record/p/11757317.html

    直接用java -jar xxx.jar，当退出或关闭shell时，程序就会停止掉。以下方法可让jar运行后一直在后台运行。

    java -jar server.jar &

    如果想要关闭java进程，输入命令行

    1，列出后台进程

    ps -ef | grep java

    2，杀死进程

    kill -9 xxxx

    (xxxx为上条命令列出的后台进程的pid， -9 表示无条件终止)
