    目前，GitHub 上最新 release 版本是 Zipkin 2.12.9，从 2.12.6 版本开始有个较大的更新，迁移使用 Armeria HTTP 引擎。

    从此版本开始，若直接添加依赖的 Spring Boot 应用启动会存在冲突，会报上面的错，降到2.12.3版本就可以正常启动了。
