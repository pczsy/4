    了解 java之前首先了解什么是软件？

    软件 是为了完成功能的程序.

    那么什么是程序？

    程序是指令的集合！

    java优势有哪些？

    java跨平台。

    java的执行过程是 我们编写的程序，编译成class的文件格式。class的文件格式，其他平台只要具有相应的解释器即可。

    java具有垃圾回收机制，而c++没有

    java是面向对象的既有继承、封装、多态。

    学大数据java是基础，因为hadoop是用java写的

    数据类型

    java的数据类型分为基本数据类型和引用数据类型

    这里面基本数据类型又分为整数型和浮点型，字符类型和boolean行型。

    整数型：byte  1字节、short  2字节、int 4字节、long 8字节

    浮点型：double  8字节、float  4字节

    字符型：char 2字节

    布尔型：trun  and false

    什么是变量？

    变量就是申请一个内存来储存值，也可以反过来说创建变量的时候要申请内存空间。

    内存管理系统会根据内存的数据类型划分存储空间，分配的空间只能用来存储该类型的数据。

    java和C一样，具有数组，流程控制语句，运算符，循环等，这里不做概述

    下面主要讲面向对象。

    什么是面向对象？

    java里面有这样一句话叫做万事万物皆对象。那么这句是什么意思呢？《java编程思想》中这样解释：将对象视为奇特的变量，他可以存储数据，除此之外，你还可以要求他自身具有执行操作。理论上讲可以讲任何待求解的问题的任何概念化构件，将其便是为程序中的对象。

    简单算法：

    package com.test.day01;



public class Test {
    public static void main(String[] args) {
        // 冒泡排序
        //冒泡排序原理 相邻的两个比较排序
        int [] arr = {3,5,77,8,34,67,4,22,66,8,9,6,45,3,5};
        int temp;
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr.length-1; j++) {
                if(arr[j]>arr[j+1]){
                    temp = arr[j];
                    arr[j]=arr[j+1];
                    arr[j+1]=temp;
                }
            }
        }
        for (int i : arr) {
            System.out.println(i);
        }
        
}
}
